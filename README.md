# To-Do List (Python)

A simple command-line To-Do List application built with Python.

## Features
- Add new tasks
- Remove existing tasks
- View all tasks
- Menu-driven interface
- User-friendly interaction

## Technologies Used
- Python

1. Clone this repository.
2. Open the project folder.
3. Run:

python application.py
